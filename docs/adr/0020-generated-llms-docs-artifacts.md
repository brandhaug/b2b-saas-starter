# Generated LLM docs artifacts

The starter ships `llms.txt` and `llms-full.txt` describing the public knowledge surface so AI coding tools can inspect the repository quickly. Today these live as checked-in static files under `apps/web/public/` and are maintained by hand; deriving them — plus a sitemap and search-index artifacts — from the MDX content at build time is a documented follow-up, not yet wired. The goal is to keep the docs, MCP, REST API, and assistant surfaces aligned around the same checked-in content.
