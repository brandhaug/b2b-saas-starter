# Effect v4 application backbone

Effect v4 is the application backbone across server, domain, infrastructure, and typed HTTP contracts. Capabilities, the API worker, the background worker, and the public `HttpApi` contract are all expressed in Effect. The React UI stays idiomatic React/TanStack for rendering; how the web app reads and mutates capability data is recorded in [client server-state](./0040-effect-atom-for-server-state.md).
