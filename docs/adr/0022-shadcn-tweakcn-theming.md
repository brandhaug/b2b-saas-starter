# shadcn and TweakCN theming

The starter uses shadcn/ui components, Tailwind CSS v4 theme tokens, and light/dark mode from the start. The current theme should reference its TweakCN source when known, and docs should point builders to TweakCN for exploring alternate shadcn-compatible themes while keeping the implementation on semantic shadcn tokens rather than raw one-off colors. The app exposes light/dark mode only; broader theme customization is documented as a code-level workflow, not an in-app theme builder.
