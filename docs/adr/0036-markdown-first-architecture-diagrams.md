# Markdown-first architecture diagrams

The starter documents its system map in `ARCHITECTURE.md`, currently as a checked-in ASCII topology diagram (a fenced `text` block) plus prose covering the auth, REST/MCP, background work, email, and webhook delivery flows. Diagrams stay as plain checked-in Markdown — ASCII art today, with Mermaid welcome for individual flows where it adds clarity — rather than a live diagramming subsystem.
